Available Routes:

Crime

provides the number of reported violent crimes in each state in the year 2020

Happiness

provides the data of which states are happiest and each states rank in 4 metrics used to calculate happiness

Income

provides average household income of each state

Park info

All Parks

provides data for parks in each state split up according to different types of parks (each state has own classifications)

State Parks

provides number of state parks in each state

Pets

provides percentage of households that own pets, cats, and dogs for each state
